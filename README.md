# Workforce & Labor Market Analytics Pipeline

**Independent Research Project | Python, SQL | Sept – Oct 2025**

## Overview
End-to-end data pipeline merging employer benefits data across Fortune 500 companies 
with US household survey records covering 45.8M+ employees. Built to produce 
executive-ready analytics outputs across health, retirement, dental, and life insurance KPIs.

## What This Project Does
- **Task 1:** Merges employer benefits + household survey datasets via entity resolution
- **Task 2:** City-level data validation and match verification across records
- **Task 3:** Automated literature review pipeline for economic research synthesis
- **Output:** Dashboard-ready benefits summary statistics for executive-level review

## Tech Stack
Python | Pandas | NumPy | SQL | CSV Pipeline Automation
